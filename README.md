# Markdown Basics:

## Formatierung (Titel)
	# Haupttitel
	## Untertitel
	### Unter-Unter-Titel

## Textformatierung
	** Fett Formatierter Text **
	* Kursiver Text *
	` Formatierung als Code `
	``` Mehrzeiliger Codeblock
	Zeile 2 ```
	> Block Quote (Zitat)

## Listen
	Geordnete Liste
		1. Eins
		2. Zwei
		3. Drei
	Ungeordnete Liste
		- Eins
		- Zwei
	Tasklists
		- [x] Erledigter Task
		- [ ] Unerledigter Task
